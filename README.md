# CQRS + Asp.Net Core
````
CQRS stands for "Command and Query Responsiblity Segregation"
Where segregation means separation.
````

## Features
- Commands and Handlers
- Domain Models (Aggregates)
- Aggregate Repository
- Event Sourcing
- Projections (Event handlers)
- ReadModels
- Query Controllers
